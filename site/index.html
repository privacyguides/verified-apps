This project is a work-in-progress, provided for informational/educational purposes only. Please discuss this project on our forum if you have any questions, and submit an app if one you have is not listed! For information about how we obtained this data, read our project README.

  • AppVerifier string: This can be copied directly into AppVerifier, or used with certhashviewer or apksigner verify --print-certs
  • Package source: Where we found the APK which was signed with this signing key. It is common for apps to use different signing keys in different app stores (Google Play and F-Droid in particular will often re-sign apps with their own keys), so if you have an app installed which does not match our list, check where you got it from!
  • More info: We keep details about where/how information was verified in our GitHub repo, as well as any user-provided notes in the comments. If you want more details about why/how an app was added to this list, check there first!
  • Tested-file SHA-256 hash: We keep this for reference to show which APK file we tested to obtain this result. It is very unlikely the APK file you have will match this, because the file hash will change with every app update.
